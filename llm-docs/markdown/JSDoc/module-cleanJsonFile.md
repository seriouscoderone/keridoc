# Module: cleanJsonFile

Cleans a JSON file by removing non-printable characters from string values.

##### Parameters:

| Name | Type | Description |
| --- | --- | --- |

| `inputFilePath` | string | The path to the input JSON file. |
| `outputFilePath` | string | The path to write the cleaned JSON data. |
